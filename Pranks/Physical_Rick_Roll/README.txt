  Rubber Ducky payload which adds a pyhsical copy of Rick Astley's Greatst Hits CD to the targets amazon basket.
  
  Requires the target to be logged into amazon, with 'remember me' enabled. Most people are.
  
  Currently configured for amazon.co.uk (British) - requires some minor changes for usage on amazon.com (American). These are simple, and highlighted in the script.
